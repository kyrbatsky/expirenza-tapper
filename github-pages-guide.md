# Публікація гри на GitHub Pages — покрокова інструкція

---

## Крок 1 — Реєстрація / вхід на GitHub

Відкрий **github.com** у браузері.

- Якщо акаунту ще немає → натисни **"Sign up"** у правому верхньому куті,
  вкажи email, пароль, username (наприклад `expirenza` або `pavel-mono`).
- Якщо акаунт є → **"Sign in"**.

> ⚠️ Username важливий: він стане частиною URL твоєї гри.
> Наприклад, username `expirenza` → гра буде на `expirenza.github.io/...`

---

## Крок 2 — Створити новий репозиторій

1. Після входу — у правому верхньому куті натисни **"+"** (плюс)
2. У меню вибери **"New repository"**

На сторінці створення заповни:

| Поле | Що вводити |
|---|---|
| **Owner** | твій username (вже стоїть автоматично) |
| **Repository name** | `expirenza-tapper` |
| **Description** | (необов'язково) `Expirenza restaurant game` |
| **Public / Private** | вибери **Public** ← обов'язково для безкоштовного Pages |
| **Add a README** | галочку **не ставити** |
| **Add .gitignore** | **None** |
| **License** | **None** |

3. Натисни зелену кнопку **"Create repository"**

---

## Крок 3 — Завантажити файли гри

Після створення репозиторію ти побачиш порожню сторінку з підказками.

1. Натисни **"uploading an existing file"** (посилання у тексті по центру сторінки)

   *(або зверху: кнопка **"Add file"** → **"Upload files"**)*

2. Перетягни у сіру зону або натисни **"choose your files"** та вибери обидва файли:
   - `expirenza-tapper.html`
   - `expirenza-tapper-iframe.html`

3. Дочекайся поки файли завантажаться (побачиш їх назви зі значком)

4. Внизу сторінки у розділі **"Commit changes"**:
   - У першому полі залиш текст як є (`Add files via upload`) або напиши `initial upload`
   - Переконайся що вибрано **"Commit directly to the main branch"**
   - Натисни зелену кнопку **"Commit changes"**

---

## Крок 4 — Увімкнути GitHub Pages

1. У репозиторії зверху знайди вкладку **"Settings"** (остання у рядку вкладок)
2. У лівому меню прокрути вниз до розділу **"Code and automation"**
3. Натисни **"Pages"**
4. У розділі **"Build and deployment"**:
   - **Source**: вибери **"Deploy from a branch"**
   - **Branch**: вибери **"main"** / папка **"/ (root)"**
   - Натисни **"Save"**

5. Сторінка оновиться. Вгорі з'явиться блок:
   > *"Your site is live at https://USERNAME.github.io/expirenza-tapper/"*

   Якщо блоку ще немає — зачекай 1–2 хвилини та оновити сторінку (F5).

---

## Крок 5 — Відкрити гру

Пряме посилання на гру:
```
https://USERNAME.github.io/expirenza-tapper/expirenza-tapper.html
```
*(замість USERNAME — твій реальний GitHub username)*

Відкрий це посилання у Chrome на Android — гра запуститься одразу.

---

## Додати на головний екран смартфона (опційно)

У Chrome на Android:
1. Відкрий посилання на гру
2. Натисни **⋮** (три крапки, правий верхній кут)
3. **"Додати на головний екран"** → **"Додати"**

Гра з'явиться як окрема іконка, схожа на додаток.

---

## Як оновити гру в майбутньому

1. Зайди у репозиторій `expirenza-tapper` на github.com
2. Натисни на назву файлу (`expirenza-tapper.html`)
3. Натисни іконку олівця ✏️ (правий верхній кут) → **"Edit"**... 

   *або простіше:* натисни **"Add file" → "Upload files"** → завантаж новий файл з тією ж назвою → GitHub замінить старий

4. Натисни **"Commit changes"** — сайт оновиться автоматично за 1–2 хв.
